> `infoq-scaffold-ui` 是基于 [plus-ui](https://github.com/JavaLionLi/plus-ui) 进行魔改，比原版更纯净，不夹带任何广告

> 后端项目地址: [infoq-scaffold](https://github.com/luckykuang/infoq-scaffold)

## 前端运行

```bash
# 安装依赖
npm install --registry=https://registry.npmmirror.com

# 启动服务
npm run dev

# 构建生产环境
npm run build:prod

# 前端访问地址 http://localhost:80
```
